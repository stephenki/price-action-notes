## 🔧 学习方式

### 自动化流程

1. **每日学习** (6:00 AM cron)
   - 通过 Hermes Agent 自动执行
   - 从 EPUB 教材提取章节内容
   - AI 生成结构化学习笔记
   - 同步更新进度追踪

2. **EPUB 来源**
   - Book 3 (Trends): `Trading Price Action - Trends (Z-Library).epub`
   - 使用 `ebooklib` 解析，提取 HTML 内容

3. **笔记格式**
   - 章节核心概念提取
   - 原文精华片段
   - 交易应用要点
   - 关联章节索引
